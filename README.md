## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/ShakalBhau0001/Python-Mini-Projects.git
```

2. **Navigate to a project folder**
```bash
cd Python-Mini-Projects/Alexa-Voice-Assistant
```

3.**Install dependencies (if applicable)**
```bash
pip install -r requirements.txt
```

4. **(Optional)** Place `DejaVuSans.ttf` in the Txt-To-PDF-Generator folder for full emoji/unicode support.

5. **Run the project**
```bash
python alexa_app.py
```
